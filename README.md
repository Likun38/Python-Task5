# Python Dictionary and List Programs

This repository contains two simple Python programs that demonstrate the use of dictionaries and lists along with basic user interaction.

---

## Program 1: Student Marks Dictionary

### Description

This program creates a dictionary where student names are stored as keys and their marks are stored as values. It prompts the user to enter a student’s name and displays the corresponding marks. If the student’s name is not found in the dictionary, an appropriate message is displayed.

### Concepts Used

* Dictionary creation
* Key-value access
* Conditional statements
* User input handling

### How to Run

```bash
python program1.py
```

---

## Program 2: List Operations

### Description

This program creates a list of numbers from 1 to 10. It extracts the first five elements from the list, reverses those elements, and prints both the extracted list and the reversed list.

### Concepts Used

* List creation
* List slicing
* List reversing
* Printing formatted output

### How to Run

```bash
python program2.py
```

---

## Requirements

* Python 3.x

These programs were developed to practice fundamental Python concepts related to data structures and basic operations.
